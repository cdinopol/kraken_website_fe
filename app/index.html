Server Info

KrakenRO is a classic super high rate, pre-renewal ragnarok server that's highly focused on MvP hunts, PvP and weekly WoE. No mindless grinding. No overpowered donates. No automated events. No weird, server-specific learning curves. It's 95% Vanilla.
